Introducción a HTML

Bienvenid@ a este tutorial de HTML donde te enseñaré las muchas cosas que puedes hacer con un simple bloc de notas.

Lo mas importante para comenzar es especificar que el tipo de documento es Html escribiendo entre etiquetas (y sin las comillas) "!doctype html".
Recuerda, deberás poner prácticamente todo entre etiquetas para que HTML lo reconozca.
Además, gran parte de las etiquetas deberán cerrarse con un / delante de la letra en la etiqueta. De no ser así, se tratarían de elementos vacíos.

A continuación, deberás especificar el lenguaje del documento (html lang="es")


Y ya comenzaríamos con la cabecera del documento en sí.
Ésto no va a ser el título que aparecerá en grande, si no que aparecerá en la pestaña donde abriste el documento.
Para crearlo, hay que poner "head", como ya sabes, entre etiquetas.

Para introducir el título, ahora sí, el que será grande o pequeño, ya dependiendo de tí, deberás poner: "h1" para el más grande y "h6" para el más pequeño. El título que véis arriba del todo está hecho con "h1"
