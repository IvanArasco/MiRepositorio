Introducción a HTML

Bienvenid@ a este tutorial de HTML donde te enseñaré las muchas cosas que puedes hacer con un simple bloc de notas.


1. Para comenzar, debes especificar que el tipo de documento es html escribiendo <!doctype html>.
2. Deberás especificar el lenguaje del documento <html lang="es"> en este caso, en español.
3. Deberás poner prácticamente todo entre etiquetas < >.
4. Gran parte de las etiquetas deberán cerrarse con un </elemento>. De no ser así, se trataría de un elemento vacío.
De momento, en HTML hay una estructura basada en una cabecera y en un cuerpo.
- La cabecera es para lo referente al documento, sin ser lo que contendrá dentro de éste.
- El cuerpo es para el contenido que tendrá el documento.

1. Cabecera:

Comenzaremos con la cabecera del documento en sí.
Hay que poner para empezar: <head>.

1.1. "Títulos":

-Ésto no va a ser el "título grande", si no lo que aparecerá en la pestaña en la que abriste el documento.
Se introducirán con la etiqueta <title>..

Ejemplos:

<title> Título Documento </title> .

2. Cuerpo:

A continuación, crearemos el "cuerpo" del documento con la etiqueta <body>
Antes de ello, deberemos cerrar head: </head>

1.1. "Headers":

-Los "headers" son para introducir el título, ahora sí, el que se visualizará en el documento y podrá ser grande o pequeño, deberás poner: <h1> para el más grande y <h6> para el más pequeño.

Ejemplos:

<h1> Título 1 </h1>

<h2> Título 2 </h2>

<h3> Título 3 </h3> <h3>

<h4> Título 4 </h4>

<h5>Título 5 </h5>
<h6>Título 6 </h6>

1.2. "Saltos de línea":

-Los saltos de línea son muy útiles para que en html no te aparezca todo en la misma línea.

Para ésto, será necesaria la etiqueta <br> para introducir el salto sin necesidad de cerrarse.
Recordemos que eso significa que es un elemento vacío.

Ejemplos:

Ésto <br>
Está <br>
Separado <br>
Por <br>
Saltos <br>
de Línea<br>

1.3. "Párrafos":

-Los párrafos son herramientas muy útiles para separar el contenido deseado.

Para ello, usaremos la etiqueta <p> y </p> , para cerrar.

<p> Contenido </p>

<p> Contenido 2 </p>

<p> Contenido 3 </p>

1.4. "Listas":

- Una lista puede ser ordenada o desordenada. Se explica mejor con ejemplos este apartado:
    <ol> Para comenzar la lista ordenada (enumerada)
  1. <li>elemento de la lista </li>
  2. <li>elemento de la lista </li>
  3. <li>elemento de la lista </li>
</ol> Para cerrar la Lista ordenada
    <ul> Para comenzar la lista desordenada
  • <li>elemento de la lista </li>
  • <li>elemento de la lista </li>
  • <li>elemento de la lista </li>
</ul> Para cerrar la Lista ordenada
- También existen las etiquetas <dl> <dd> <dt>
Pero lo esencial para las listas, son las etiquetas mencionadas anteriormente.

1.5. "Hipervínculos":

- Un hipervínculo sirve principalmente para moverte a otra parte del documento o a otro sitio.
<a href="link"> Texto del hipervínculo </a>

Ejemplos:

Con este hipervínculo vas a google

1.6. "Imágenes":

- Una imágen puede añadirse con un montón de características que uno quiera
Como ponerla en un punto del documento y dejarla fija,
Como establecer un tamaño para ésta
- No obstante, necesitaremos conocer la dirección dónde se aloja dicha imágen y ponerla.
<img src="imagen.jpg" alt="Comentario" >
- También puedes establecer el tamaño con <width> y <height>
<img src="imagen.jpg" alt="Comentario" width=50% height=20% >

Ejemplos:

(Con Width 30% y height 20%)

1.7. "Tablas":

- Una tabla tiene varias aplicaciones, como por ejemplo, crear un horario.
<table>
<tr> </tr> Las filas de la tabla.
<td> </td> Las columnas de la tabla.
<th> </th> Título de la tabla.
<thead> </thead> Encabezado de la tabla.
<tbody> </tbody> Contenido de la tabla.
</table> - También podemos incluir "bordes" poniendo en la etiqueta <table>
<table border=nº>(recomendado el 1)

Ejemplos:

Título de la tabla
Columna1
Columna2
Columna3 Columna fuera

1.8. "Etiquetas adicionales":

- Las etiquetas "adicionales" sirven para darle cierto potencial visual al documento.
Por ejemplo, si queremos resaltar algo en negro, subrayarlo, tacharlo...
Serán necesarias las etiquetas:

<b> Texto resaltado </b>
<u> Texto Subrayado </u>
<s> Texto Tachado </s>
